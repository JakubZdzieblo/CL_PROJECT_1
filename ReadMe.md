### Coders Lab Project 1

"Car Service" CRM app made for 4th week of Java bootcamp.

Technologies / models used:

 - Java servlets
 - JSTL
 - JSP
 - MVC
 - Bootstrap
 - DAO
 - MySQL

Functions:

 - CRUD for Employees, Customers, Vehicles and Orders
 - Generating reports
